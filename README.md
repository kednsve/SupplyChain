#  供应链优化分析平台

## 1. 项目概述

- **背景**：供应链管理中普遍存在**库存成本过高、物流时效低、供应商绩效不透明**等痛点，传统经验决策难以精准优化。
- **目标**：构建一个**数据驱动、全栈实现**的供应链优化分析平台，通过数据采集、分析建模、可视化展示，实现：
  - 库存需求精准预测（降低库存积压/缺货风险）；
  - 物流节点效率分析（识别瓶颈，优化路径）；
  - 供应商多维度评估（筛选优质供应商）；
  - 实时监控与预警（如库存阈值、物流延迟）。

## 2. 技术栈

| 层级           | 技术                         |
| -------------- | ---------------------------- |
| 后端服务       | SpringBoot + MyBatis         |
| 前端页面       | Vue3 + Element Plus          |
| 数据分析与预测 | Python (Pandas/Scikit-learn) |
| 可视化         | Echarts                      |
